<link rel="stylesheet" href="css/index.css"> 

link - тег для подключения файлов
rel - атрибут, который задает содержимое файла
stylesheet - значение атрибута, указывает, что подключаемый файл это таблица стилей
href - атрибут, который указывает путь к файлу
css/index.css - значение атрибута href. [css/ - папка, index.css - файл В папке css/]
